# Suhinsmi Website

A responsive personal portfolio website built with plain HTML, CSS, and JavaScript.

## How to use

1. Open `index.html` in any web browser.
2. Edit the text inside `index.html`.
3. Change colors in the `:root` section of `styles.css`.
4. Replace `hello@example.com` with your real email address.
5. Replace the sample projects and links with your own work.

## Publish it free

You can upload these files to:
- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

No build tools are required.
